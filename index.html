Description

Bot Basique Discord en Javascript. Marche avec discord.js ainsi que Node.
Un simple Bot Discord utilisable pour commencer à développer le sien.

settings Création et ajout du Bot Discord

Créer un nouveau Bot Discord sur Discordapp :

  • Aller sur la page : https://discordapp.com/developers/docs/intro
  • Puis MyApplications > New Application.
  • Complétez le formulaire. Une fois fait, cliquez sur "Créer Application".
  • Cliquez sur "Créer un utilisateur Bot", puis cliquez sur "Oui, faites-le!"
  • Maintenant, ajoutez votre bot à votre Serveur Discord. Rappel : Remplacer "[ID_Bot]" avec votre bot client ID.

Pour les permissions , je vous conseille d'aller sur Permissions Calculator.
Maintenant que vôtre Bot est créer et qui à rejoint vôtre serveur, suivez le prossessus pour le développer et l'exécuter.

file_download Installation

Requiert : node 6.9.1 ( Dernière version ) Télécharger la dernière version de NodeJS.

Veuillez mettre en place un nouveau dossier pour tout vos scripts JS et faire les commandes dans celui donné.

npm install --save discord.js

Une liste d'autres bibliothèques d'API Discord peut être trouvée ici. Après avoir tout fait, vous pouvez l'exécuter depuis vôtre invité de commande ou se trouve vôtre dossier de script pour le bot avec un coup de node [fichier.js].
