Basic Bot Discord

Description

Basique Bot Discord en Javascript. Marche avec discord.js ainsi que Node.
Un simple Bot Discord utilisable pour commencer à développer le sien.

settings Création et ajout du Bot Discord

Tout d'abord , créer un nouveau Bot Discord sur Discordapp :

  • Aller sur la page : https://discordapp.com/developers/docs/intro
  • Puis MyApplications > New Application.
  • Complétez le formulaire. Une fois fait, cliquez sur "Créer Application".
  • Cliquez sur "Créer un utilisateur Bot", puis cliquez sur "Oui, faites-le!"
  • Maintenant, ajoutez votre bot à votre Serveur Discord. Rappel : Remplacer "[ID_Bot]" avec votre bot client ID.
    • https://discordapp.com/oauth2/authorize?client_id=[ID_Bot]&scope=bot&permissions=0

    Pour les permissions , je vous conseille d'aller sur Permissions Calculator.
    Maintenant que vôtre Bot est créer et qui à rejoint vôtre serveur,
    suivez le prossessus pour le développer et l'exécuter.
